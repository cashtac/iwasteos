# iWaste

Camera-first AI sustainability assistant that reduces food waste and improves recycling accuracy.

## Live Demo

**https://iwaste.daniil-osipov.com**

## Current Features

- Food scanning simulation
- Waste sorting assistant
- Eco scoring engine
- Session-based sustainability tracking

## Architecture (Current)

- Static frontend (HTML / CSS / JS)
- Local state management
- Designed for future cloud API integration

## File Structure

```
iwasteos/
├── index.html    — Landing page
├── style.css     — Visual system
├── app.js        — Demo logic + scanner controller
└── assets/       — Static assets
```
